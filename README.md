## Getting started ğŸ› 

- Copy app

```
$ git clone https://github.com/veronga/game
```

- Open downloaded folder

```
$  cd game
```

- Install necessary packages

```
$  npm install
```

- Install PODs

```
$  cd ios
```

```
$  pod install
```

- - Now let's launch the app, but don't forget to return to the main folder

```
$  cd ..
```

```
$  react-native run-[ios|android]
```
